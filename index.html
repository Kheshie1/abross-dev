🚀 Installation

One-Command Install

curl -sSL https://raw.githubusercontent.com/Kheshie1/abross-dev/main/install.sh | bash

Or using Cargo:

cargo install abross-ai-terminal

Build from source:

git clone https://github.com/Kheshie1/abross-ai-terminal
cd abross-ai-terminal
cargo build --release
./target/release/abross-ai-terminal
