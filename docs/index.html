Abross Agentic Development Environment

🚀 Installation

cargo install abross-ai-terminal

Or build from source:

git clone https://github.com/Kheshie1/abross-ai-terminal
cd abross-ai-terminal
cargo build --release
./target/release/abross-ai-terminal

⚡ Quick Start

Once installed, simply run:

abross-ai-terminal

Press Ctrl+H for help at any time.

⌨️ Keyboard Shortcuts

Ctrl+A - Toggle AI mode
Ctrl+T - New tab
Ctrl+W - Close tab
Ctrl+R - Rename tab
Ctrl+Tab - Next tab
Ctrl+Shift+Tab - Previous tab
Ctrl+F - Search history
Ctrl+S - Split pane
Ctrl+Shift+S - SSH connect
Ctrl+P - Time travel
Ctrl+H - Help panel
Tab - Accept AI suggestion
↑/↓ - Command history
Esc - Cancel command
Ctrl+C - Quit

🤖 AI Features

With AI mode ON (Ctrl+A), try:

list files                    → ls -la
show disk usage              → df -h
show memory                  → free -h
find rust files              → find . -name '*.rs'
show git status              → git status
create an EC2 instance       → AWS EC2 command
build a docker image         → docker build -t my-app .
terraform plan               → terraform plan

🔌 SSH Support

Connect to remote servers:

ssh://username@hostname:port
Then press Ctrl+Shift+S to connect

☁️ Infrastructure as Code

Abross supports multiple cloud providers:

• AWS EC2, Lambda, S3, RDS, EKS
• Kubernetes (Deployment, Service, Ingress)
• Docker (Dockerfile, Docker Compose)
• Terraform (init, plan, apply)

🔌 Plugins

Install plugins from the marketplace:

abross plugin install git-flow
abross plugin list
abross plugin update

🎨 Themes

Change themes in ~/.config/abross/config.toml:

theme = "catppuccin"
# Available: dark, light, catppuccin, nord, tokyo-night

🔧 Troubleshooting

Terminal not responding? Press Esc or Ctrl+C

AI not working? Set OPENAI_API_KEY environment variable

SSH connection fails? Check your SSH keys and network
